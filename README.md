## Specifications

### Tooling

This project now uses Vite for local development and production builds.

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- npm v9+

### Install dependencies

```bash
npm install
```

### Dev server

```bash
npm run dev
```
Starts Vite at `http://localhost:5173` and opens the browser automatically.

### Production build

```bash
npm run build
```
Outputs optimised assets to `dist/`.

### Preview production build locally

```bash
npm run preview
```

### Day 02 | 4/12/2026 - Sunday

Asking ChatGPT to help me write an about me page that talks about my cats.

The Cat Window is a collection of my pictures of the 6 cats I've owned over my lifetime. A short about my cats section that shows who they were. Blog posts that show rememberance. A shop that will have affiliate links to products required by the cat owner. Articles from other sites that shed light on what your cat does. Random facts about cats for trivia night. Also there is an event list that will show adoptions, shows, and hours of local cat cafes.

Current two are Barnacle and Bean.
Before them Kashmir and Dante.
First we had Laser and Figment.

The first two were a black cat and tuxedo. Following them it was two tuxedo cats. Now we have a tabby and calico.

Laser and Figment were in my life before I was born! They would be a year older than me, making them 37 and 38 if they were alive. We had them since kittens.

Kashmir and Dante were both rescue cats. Kashmir from a rescue foundation where he was standoff-ish except for me. We trained him to calm down and enjoy house life. Dante was found outside. He was just a kitten. We took him in and he was so sweet and a vocal cuddlebug.

Barnacle and Bean are shelter cats. They are 3 and 4 years old now. They are full of personality, especially Barnacle, who will attack you for food when he says it is meal time. Bean loves her water from the sink. She manages to always find a way to snag a drink from the fountain. She also picks bathroom time to enjoy my company.
